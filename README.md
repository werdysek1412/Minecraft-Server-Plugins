Minecraft Server Plugins
A collection of custom, optimized plugins for Minecraft servers.

📁 Included Plugins
ServerNPC: An interactive NPC system for teleportation, messages, and rewards.

WorldGuard: Professional region management with flag and priority support.

Auth: A system for player authentication and security.

LobbySystem: Management for lobbies and world-specific respawns.

MineQuarry: An advanced automated mining system.

Residence: Protection for player plots and residences.

🚀 Installation
Upload the desired .jar file to your /plugins folder.

Restart your server using the /stop command.

The plugins are ready for use.

🛠 Usage and Sharing
These plugins are designed for administrators and server owners. All tools support permissions management and are built for professional-level operation. This project is freely available—you are welcome to share, modify, and use the code for your server needs.
