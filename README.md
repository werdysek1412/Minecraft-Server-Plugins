Minecraft Server Plugins
A collection of custom, optimized plugins for Minecraft servers.

Included Plugins
ServerNPC: Interactive NPC system for teleportation and rewards.

WorldGuard: Region management, flags, and priority settings.

Auth: Player authentication and security.

LobbySystem: Management for lobbies and world-specific respawns.

MineQuarry: Automated mining system.

Residence: Protection for player residences.

Installation
Upload the .jar file to your /plugins folder.

Restart the server using the /stop command.

Usage and Sharing
These plugins are intended for administrators and server owners. All tools support permissions management. You are free to share, modify, and use the code for your server needs.
